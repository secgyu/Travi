## 해외 사용 카드 종류

### 신용카드

**장점**

- 해외 여행자 보험 자동 가입
- 포인트 적립
- 긴급 현금 서비스
- 렌터카 보증금 결제 가능

**단점**

- 연회비 부담
- 과소비 위험
- 해외 사용 수수료

### 체크카드

**장점**

- 연회비 무료 많음
- 과소비 방지
- 환전 우대율 적용

**단점**

- 여행자 보험 미적용
- 렌터카 보증금 불가능한 경우 있음
- 포인트 적립 적음

### 선불카드

**장점**

- 환율 고정 가능
- 과소비 방지
- 미성년자도 사용 가능

**단점**

- 충전 번거로움
- 사용처 제한
- 잔액 환불 수수료

## 해외 사용 수수료

### 수수료 구조

**해외 사용 수수료**

- VISA/Master: 1%
- JCB: 1.1%
- 카드사 수수료: 0.1-0.2%
- 총: 약 1.2-1.3%

**ATM 인출 수수료**

- 카드사 수수료: 3,000-5,000원
- 현지 ATM 수수료: $2-5
- 총: 5,000-10,000원

### 수수료 아끼는 법

- 환전 우대 카드 사용
- DCC(원화 결제) 거부
- ATM 인출 최소화
- 마스터/비자 우선 사용

## 추천 카드

### 신용카드

**하나카드 1Q5**

- 해외 가맹점 5% 캐시백
- 연회비: 국내 1만원, 해외 2만원
- 여행자 보험 최대 1억원

**현대카드 M3**

- 해외 가맹점 5-7% 할인
- 연회비: 3만원
- 공항 라운지 무료

**삼성카드 탭탭 O**

- 해외 가맹점 3-5% 할인
- 연회비: 국내 무료, 해외 2만원
- 교통/통신비 할인

### 체크카드

**카카오뱅크 체크카드**

- 해외 ATM 인출 수수료 면제 (월 5회)
- 연회비: 무료
- 환전 우대 90%

**토스뱅크 체크카드**

- 해외 결제 수수료 환급
- 연회비: 무료
- 실시간 환율 적용

**우리은행 위비 체크카드**

- 환전 수수료 우대
- 연회비: 무료
- ATM 수수료 면제

## DCC 거부하기

### DCC란?

⚠️ Dynamic Currency Conversion의 약자로, 현지 통화가 아닌 원화로 결제하는 방식입니다.

**문제점**

- 환율이 5-10% 높음
- 추가 수수료 부과
- 손해가 큼

### 거부 방법

결제 시 다음과 같이 말하세요:

- "Please charge in local currency"
- "No, charge in (USD/EUR/JPY)"
- "I don't want DCC"

**카드 단말기에서**

- "원화(KRW)" 선택하지 말기
- "현지 통화" 선택
- 의심스러우면 취소 후 다시 결제

## 카드 분실/도난 대처법

### 즉시 할 일

1. 카드사에 전화 (분실 신고)
2. 카드 사용 정지
3. 경찰서에 신고서 제출
4. 대사관/영사관 연락

### 긴급 연락처

카드사별 해외 긴급 연락처를 메모해두세요.

**주요 카드사**

- 신한카드: +82-2-6933-6005
- 삼성카드: +82-2-2000-8100
- 현대카드: +82-2-3015-9000
- KB국민카드: +82-2-6300-8500

### 재발급

- 긴급 재발급 서비스 이용
- 2-3일 내 호텔로 배송
- 수수료: 5-10만원

## 안전한 카드 사용

### 보안 팁

💡 카드 안전 사용 팁:

- 카드 뒷면 서명하기
- CVV 번호 가리기
- 비밀번호는 생년월일 피하기
- 이상 거래 즉시 신고

### 사용 시 주의사항

- 카드가 눈앞에서 결제되는지 확인
- 영수증 보관
- 명세서 정기 확인
- 소액 결제도 영수증 확인

### 스키밍 방지

- ATM 사용 전 카드 투입구 확인
- 의심스러운 ATM 피하기
- 가능하면 은행 내부 ATM 사용
- 비밀번호 입력 시 가리기

## 환율 확인 방법

### 실시간 환율 앱

- 하나은행 하나원큐
- 네이버 금융
- 구글 환율 계산기

### 환율 팁

- 업무일 오전이 환율 좋음
- 주말/공휴일 환율 불리
- 카드사 고시 환율 확인
- 환율 우대 쿠폰 활용
