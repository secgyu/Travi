## ESTA란?

ESTA(Electronic System for Travel Authorization)는 비자 면제 프로그램으로 미국을 방문하는 한국인이 비자 없이 입국할 수 있도록 하는 전자 여행 허가제입니다.

### ESTA vs 비자

**ESTA 적용 대상**

- 관광, 출장 목적
- 90일 이하 체류
- 왕복 항공권 소지
- 비자 면제 협정국 국민

**비자가 필요한 경우**

- 90일 초과 체류
- 유학, 취업 목적
- 과거 입국 거부 경력
- 범죄 경력

## 신청 방법

### 1. 공식 사이트 접속

**주의: 반드시 공식 사이트 이용**

공식 사이트: https://esta.cbp.dhs.gov

- 수수료: $21 (약 28,000원)
- 사칭 사이트 주의 (수수료 과다 청구)
- .gov 도메인 확인

### 2. 필요한 정보 준비

**개인 정보**

- 여권 정보 (번호, 발급일, 만료일)
- 영문 이름 (여권과 동일하게)
- 생년월일
- 국적
- 주소
- 전화번호
- 이메일

**여행 정보**

- 미국 내 숙소 주소
- 비상 연락처
- 여행 목적
- 항공편 정보 (선택사항)

### 3. 신청 절차

1. 언어 선택 (한국어 가능)
2. 개인 정보 입력
3. 여권 정보 입력
4. 여행 정보 입력
5. 적격성 질문 답변 (9개 질문)
6. 정보 확인
7. 결제 ($21)
8. 승인 대기

### 4. 승인 확인

**처리 시간**

- 일반적으로 즉시~72시간
- 대부분 수분 내 승인
- 여유있게 출발 2주 전 신청 권장

**승인 상태**

- Authorization Approved (승인)
- Travel Not Authorized (거부)
- Pending (대기 중)

## 적격성 질문

### 9가지 질문

모두 "아니오"로 답변해야 합니다.

1. 전염병 보유 여부
2. 정신/신체 장애 여부
3. 마약 관련 범죄 경력
4. 체포/유죄 판결 경력
5. 비자 거부/입국 거부 경력
6. 미국 불법 체류 경력
7. 테러 활동 참여
8. 과거 미성년자 양육권 박탈
9. 다른 사람을 대신해 신청하는지 여부

### 주의사항

- 솔직하게 답변
- 거짓말은 영구 입국 금지
- 경미한 교통 위반은 "아니오"
- 의심스러우면 대사관 상담

## 유효기간 및 연장

### 유효기간

**ESTA**

- 승인일로부터 2년
- 또는 여권 만료일까지

**체류 기간**

- 1회 최대 90일
- 연장 불가
- 90일 초과 시 재입국 금지 가능

### 재신청이 필요한 경우

- ESTA 만료 시
- 여권 재발급 시
- 이름 변경 시
- 성별 변경 시
- 국적 변경 시
- 적격성 질문 답변 변경 시

## 입국 심사

### 준비 서류

- 여권
- ESTA 승인 (출력본 또는 확인번호)
- 왕복 항공권
- 숙소 예약 확인서
- 여행 일정표
- 재정 증빙 자료

### 질문 예시

**기본 질문**

- "What's the purpose of your visit?" (방문 목적)
- "How long will you stay?" (체류 기간)
- "Where will you stay?" (숙소)
- "What do you do?" (직업)

**추가 질문**

- 동행자 유무
- 이전 미국 방문 경력
- 귀국 계획

### 답변 요령

- 짧고 명확하게
- 관광/출장 목적 강조
- 귀국 의지 표현
- 당황하지 말기
- 영어 못해도 OK (통역 요청)

## 거부 사유

### 일반적인 거부 이유

- 범죄 경력
- 과거 불법 체류
- 비자 규정 위반
- 부정확한 정보 제공
- 이란, 이라크, 시리아 등 방문 경력

### 거부 시 대처법

1. 거부 사유 확인
2. 주한 미국 대사관 방문
3. 비자 신청 (B-1/B-2)
4. 인터뷰 예약
5. 필요 서류 준비

## 비용

### ESTA

- 신청 수수료: $21
- 유효기간: 2년
- 무제한 입국 (1회 90일)

### 대행업체

- 수수료: 3-10만원 추가
- 필요성: 낮음 (직접 신청 권장)
- 주의: 사칭 사이트 많음

## 체크리스트

신청 전 확인사항:

- ✓ 여권 유효기간 6개월 이상
- ✓ 공식 사이트 확인 (.gov)
- ✓ 여권 정보 정확히 입력
- ✓ 이메일 주소 올바르게 입력
- ✓ 신청 번호 저장
- ✓ 승인 확인서 출력
- ✓ 출발 2주 전 신청
- ✓ 결제 준비 ($21)

## 유용한 팁

- 여러 명 동시 신청 가능 (그룹 신청)
- 승인 후 여행 계획 변경 가능
- 항공편 정보 입력 선택사항
- 모바일로도 확인 가능
- 승인 번호만 있으면 출국 가능
- 승인 상태는 웹사이트에서 재확인
