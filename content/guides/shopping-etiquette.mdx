## 해외 쇼핑 기본 에티켓

나라마다 쇼핑 문화가 다릅니다. 실수하지 않도록 기본 에티켓을 알아두세요.

### 공통 에티켓

- 상품을 함부로 만지지 않기 (특히 고가품)
- 직원 허락 받고 만지기
- 시식/시향 후 구매 강요 없음
- 옷은 반드시 원래 위치에
- 계산 줄 지키기
- 큰 소리로 말하지 않기

## 나라별 쇼핑 문화

### 일본

**특징**

- 극도로 친절한 서비스
- 상품 포장 정성스러움
- 계산 시 트레이 사용
- 만지기 전 허락 구하기

**에티켓**

- 계산대에서 돈은 트레이에
- 직원이 포장할 때까지 기다리기
- "아리가또 고자이마스" (감사합니다)
- 상품 훼손 시 반드시 구매

**쇼핑 시간**

- 오전 10시~저녁 8시
- 백화점: 10시~8시 30분
- 편의점: 24시간

### 한국

**특징**

- 동대문, 명동 등 쇼핑 천국
- 시식 문화 발달
- 흥정 가능 (시장)
- 면세점 많음

**에티켓**

- 시장에서 흥정 OK
- 백화점/브랜드숍은 정가
- 시식 마음껏 (구매 압박 없음)
- 환불 비교적 자유로움

### 미국

**특징**

- 셀프 서비스
- 환불 정책 관대
- 세일 많음
- 쿠폰 문화

**에티켓**

- "Excuse me" 먼저 말하기
- 옷은 꼭 원위치
- 탈의실에 들고 들어간 옷은 밖 랙에
- 계산 시 인사 나누기

**환불**

- 영수증 있으면 대부분 환불 가능
- 30-90일 이내
- 너그러운 편

### 프랑스

**특징**

- 상점 문화 보수적
- 직원에게 먼저 인사
- 고급 브랜드 많음
- 예의 중요

**에티켓**

- 들어갈 때 "Bonjour" (봉쥬르)
- 나갈 때 "Au revoir" (오 흐부아)
- 직원 허락 없이 만지지 않기
- 세일 기간 : 1월, 7월

### 영국

**특징**

- 줄 서기 문화
- 정중한 태도
- 백화점 고급스러움
- 세일 기간 많음

**에티켓**

- 반드시 줄 서기
- "Please", "Thank you" 필수
- 직원과 아이 컨택
- 계산 시 간단한 대화

### 중국

**특징**

- 흥정 문화 강함
- 가짜 상품 주의
- 큰 목소리
- 빠른 계산

**에티켓**

- 시장에서 50-70% 흥정
- 가격 확인 꼼꼼히
- 현금 선호
- 위챗페이/알리페이 보편적

### 동남아

**특징**

- 흥정 문화
- 야시장 많음
- 저렴한 가격
- 복제품 많음

**에티켓**

- 30-50% 흥정 가능
- 미소 지으며 흥정
- 여러 곳 비교
- 대량 구매 시 할인 요청

## 흥정하는 법

### 흥정 가능한 곳

- 재래시장
- 노점상
- 개인 상점
- 동남아 대부분

### 흥정 불가능한 곳

- 백화점
- 브랜드 매장
- 슈퍼마켓
- 체인점

### 흥정 방법

💡 흥정 꿀팁:

1. 가격 물어보기
2. 관심 있는 척하며 둘러보기
3. 50-70% 제시
4. 상인이 반응 보기
5. 점진적으로 올리기
6. 안 되면 가는 척
7. 불러주면 거래

**영어 표현**

- "How much?" (얼마예요?)
- "Too expensive" (너무 비싸요)
- "Give me discount" (할인해주세요)
- "Best price?" (최저가요?)
- "I'll buy 2, give discount" (2개 사면 할인?)

## 면세 쇼핑

### Tax Free

**대상 국가**

- 유럽 (VAT 환급)
- 일본 (소비세 환급)
- 싱가포르 (GST 환급)
- 한국

**환급 절차**

1. 면세 가능 매장 확인
2. 최소 구매 금액 확인
3. 영수증과 서류 받기
4. 공항에서 세관 확인
5. 환급 카운터에서 환급

**환급률**

- 유럽: 8-12%
- 일본: 8-10%
- 싱가포르: 7%

💡 팁: 신용카드 환급이 현금보다 유리

### 공항 면세점

**장점**

- 세금 없음
- 브랜드 다양
- 한도까지 구매 가능

**단점**

- 시내보다 비쌀 수 있음
- 비교 불가능
- 시간 제한

**한도**

- 주류: 1L (2병)
- 담배: 200개비 (1보루)
- 향수: 60ml
- 기타: $600 또는 $800

## 환불 및 교환

### 환불 정책 확인

**구매 전 확인**

- 환불 가능 기간
- 영수증 필요 여부
- 택 제거 여부
- 환불 방법 (현금/카드)

### 나라별 환불 정책

**미국**

- 매우 관대
- 30-90일
- 영수증 없어도 가능한 경우 있음

**유럽**

- 14일 이내
- 미사용 상태
- 택 부착

**아시아**

- 7-14일
- 까다로운 편
- 교환만 가능한 경우 많음

## 쇼핑 사기 주의

### 흔한 사기 유형

⚠️ 주의하세요:

**바가지**

- 관광지 주변 고가 판매
- 택시 기사 추천 상점
- 여러 곳 비교 필수

**가짜 상품**

- 브랜드 복제품
- 품질 확인
- 정품 인증서 요구

**강매**

- 집요한 판매
- 단호하게 거절
- 끌려 들어가지 않기

**바꿔치기**

- 계산 전후 상품 바꿔치기
- 포장 확인
- 직접 챙기기

### 안전한 쇼핑

💡 안전한 쇼핑 팁:

- 공식 매장 이용
- 영수증 반드시 받기
- 품질 확인
- 가격 비교
- 리뷰 확인
- 현지인 추천 상점

## 쇼핑 팁

### 세일 시즌

**미국**

- Black Friday (11월 넷째 주 금요일)
- Cyber Monday
- 연말연시

**유럽**

- 여름 세일 (7월)
- 겨울 세일 (1월)

**일본**

- 신년 복주머니 (1월)
- 여름 세일 (7-8월)

**한국**

- 코리아 세일 페스타 (9-10월)
- 연말 세일

### 현명한 쇼핑

💡 쇼핑 절약 꿀팁:

- 세일 시즌 활용
- 현지 브랜드 구매
- 면세점 vs 시내 가격 비교
- 환율 좋을 때
- 쿠폰 활용
- 대량 구매 할인
- 오전에 가기 (한산함)
- 평일이 주말보다 나음

## 체크리스트

쇼핑 전:

- ✓ 예산 정하기
- ✓ 쇼핑 리스트 작성
- ✓ 환율 확인
- ✓ 면세 한도 확인
- ✓ 세일 기간 확인
- ✓ 리뷰 확인

쇼핑 시:

- ✓ 여권 소지 (면세 쇼핑)
- ✓ 영수증 챙기기
- ✓ 품질 확인
- ✓ 가격 비교
- ✓ 환불 정책 확인
- ✓ 택 제거하지 않기 (환불 대비)

귀국 시:

- ✓ 세관 신고서 작성
- ✓ 면세 한도 확인
- ✓ 영수증 보관
- ✓ 신고 대상 확인
