## 여행 경비 구성

여행 경비는 크게 3가지로 나눠 준비하는 것이 안전합니다.

### 기본 구성 비율

**추천 구성:**

- 카드: 60-70%
- 현금: 20-30%
- 비상금: 10%

## 현금 준비

### 얼마나 환전할까?

**여행 기간별 추천 금액 (1인 기준)**

**동남아 (3박 4일)**

- 현금: $200-300
- 카드: $300-400
- 총: $500-700

**일본 (5박 6일)**

- 현금: ¥30,000-50,000
- 카드: 추가 지출
- 총: 약 50만원

**유럽 (7박 8일)**

- 현금: €200-300
- 카드: €500-700
- 총: 약 100-120만원

### 현금이 필요한 곳

- 재래시장, 노점상
- 소규모 상점, 식당
- 대중교통 (버스, 지하철)
- 팁 (미국, 유럽 등)
- 화장실 이용료
- 긴급 상황

### 현금 보관 방법

💡 안전한 현금 보관 팁:

- 여러 곳에 분산 보관
- 호텔 금고 이용
- 숨김 벨트, 목걸이 지갑 활용
- 큰 금액과 작은 금액 분리
- 일일 사용액만 지갑에

## 카드 준비

### 카드 조합 추천

**추천 구성:**

1. 해외 우대 신용카드 1장
2. 체크카드 1장
3. 예비 카드 1장

**분리 보관:**

- 지갑: 주 사용 카드 2장
- 짐: 예비 카드 1장

### 카드별 용도

**신용카드**

- 호텔, 렌터카 예약
- 큰 금액 결제
- 긴급 현금 인출
- 여행자 보험 혜택

**체크카드**

- 일상 소액 결제
- ATM 현금 인출
- 과소비 방지

### 출발 전 확인사항

⚠️ 반드시 확인하세요:

- 해외 사용 승인 확인
- 카드 한도 확인 및 증액
- 긴급 연락처 저장
- 유효기간 확인
- 체크카드 잔액 충분한지 확인

## 여행자 수표 (T/C)

### 여행자 수표란?

- 현금처럼 사용 가능한 수표
- 분실 시 재발급 가능
- 최근에는 잘 사용 안 함

### 장단점

**장점**

- 분실 시 재발급 가능
- 안전함
- 수수료 저렴

**단점**

- 사용처 제한적
- 환전 번거로움
- 수수료 발생
- 젊은 세대는 잘 모름

💡 참고: 카드가 보편화되면서 여행자 수표는 거의 사용하지 않습니다.

## 선불 카드

### 종류

**외화 선불카드**

- 트래블 월렛
- 트래블 페이
- 하나머니 등

**특징**

- 환율 고정 가능
- 수수료 저렴
- 충전식

### 장단점

**장점**

- 환율 변동 위험 없음
- 예산 관리 용이
- 수수료 저렴

**단점**

- 사용처 제한
- 잔액 환불 번거로움
- 발급 수수료

## 지역별 추천 구성

### 일본

**현금 위주**

- 현금: 60% (¥30,000-50,000)
- 카드: 40%
- 이유: 현금 문화, 소규모 상점 많음

### 미국/유럽

**카드 위주**

- 카드: 80%
- 현금: 20% ($100-200 또는 €100-200)
- 이유: 카드 보편화, 팁 제외하면 현금 불필요

### 동남아

**균형잡힌 구성**

- 카드: 50%
- 현금: 50%
- 이유: 소규모 상점 많지만 카드도 잘 됨

### 중국

**특별 준비**

- 위챗페이, 알리페이 필수
- 현금은 비상용만
- 한국 카드는 제한적

## ATM 현금 인출

### 해외 ATM 이용법

1. Plus, Cirrus 마크 확인
2. 카드 삽입
3. 언어 선택 (English)
4. PIN 번호 입력
5. Withdrawal 선택
6. 금액 선택
7. 영수증 받기

### 수수료

**총 수수료 구성**

- 카드사 수수료: 3,000-5,000원
- 현지 ATM 수수료: $2-5
- 총: 약 5,000-10,000원

💡 팁: 한 번에 큰 금액 인출해서 수수료 절약

### 주의사항

⚠️ ATM 사용 시 주의:

- 은행 내부 ATM 우선 이용
- 스키밍 장치 확인
- 비밀번호 입력 시 손으로 가리기
- 밤늦은 시간 피하기
- 영수증 꼭 챙기기

## 비상금 준비

### 미화 달러 준비

💡 꿀팁: 미화는 전 세계 어디서나 환전 가능

- $100 지폐 2-3장
- 여권과 별도 보관
- 긴급 상황 대비

### 비상 연락처

**저장해두기**

- 카드사 긴급 연락처
- 보험사 연락처
- 대사관/영사관 연락처
- 가족 연락처

## 환율 체크

### 환율 확인 방법

- 네이버 금융
- 하나은행 앱
- 환율 계산기 앱
- 구글 검색

### 환율 우대 받는 법

- 은행 앱에서 쿠폰 다운로드
- 우대 고객 혜택 이용
- 대량 환전 시 협상
- 인터넷 환전 이용

## 예산 관리 팁

### 일일 예산 설정

**계획 세우기**

1. 총 여행 경비 계산
2. 항공, 숙박 제외
3. 남은 금액 ÷ 여행 일수
4. 일일 예산 설정

### 지출 기록

- 가계부 앱 활용
- 매일 저녁 정리
- 카테고리별 구분 (식비, 교통비, 쇼핑 등)
- 예산 초과 시 다음 날 조절

### 비용 절약 팁

💡 여행 경비 절약 꿀팁:

- 점심은 저렴하게, 저녁은 괜찮은 곳
- 슈퍼마켓 활용 (간식, 음료)
- 무료 관광지 찾기
- 대중교통 이용
- 현지인 추천 맛집
- 불필요한 기념품 자제

## 귀국 후 처리

### 남은 현금

**선택지**

- 재환전 (수수료 높음)
- 다음 여행 보관
- 공항 기부함
- 지인에게 판매

### 카드 정리

- 명세서 확인
- 부당 청구 확인
- 포인트 적립 확인
- 연회비 발생 주의

- ✓ 카드 2-3장 준비
- ✓ 해외 사용 승인 확인
- ✓ 카드 한도 확인
- ✓ 긴급 연락처 저장
- ✓ 미화 비상금 준비
