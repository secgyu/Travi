## 모바일 결제란?

스마트폰을 이용해 카드 없이 결제하는 방법입니다. 해외 여행 시 지갑 없이도 편리하게 결제할 수 있습니다.

## 주요 모바일 결제 서비스

### Apple Pay (애플페이)

**지원 기기**

- 아이폰 (6 이상)
- 애플워치
- 아이패드
- 맥북

**사용 가능 국가**

- 미국, 영국, 프랑스, 일본, 호주 등 70개국 이상
- 한국은 현대카드만 가능

**장점**

- 빠르고 간편
- 보안 강력 (Face ID, Touch ID)
- NFC 단말기만 있으면 사용 가능

### Google Pay (구글페이)

**지원 기기**

- 안드로이드 스마트폰
- 안드로이드 워치

**사용 가능 국가**

- 미국, 영국, 호주, 싱가포르 등 40개국 이상
- 한국 카드 등록 가능 (일부 카드사)

**장점**

- 안드로이드 사용자 필수
- 교통카드 기능
- 포인트 통합 관리

### Samsung Pay (삼성페이)

**지원 기기**

- 갤럭시 스마트폰
- 갤럭시 워치

**특징**

- MST 기술로 구형 단말기도 사용 가능
- 해외에서도 한국 카드 사용

**사용 가능 국가**

- 미국, 스페인, 중국, 인도 등
- 한국 카드로 해외 결제 가능

## 설정 방법

### Apple Pay 설정

1. Wallet 앱 실행
2. 카드 추가 (+버튼)
3. 카드 정보 입력 또는 카메라로 스캔
4. 카드사 인증 (SMS, 앱 등)
5. 완료

### Google Pay 설정

1. Google Pay 앱 다운로드
2. 카드 추가
3. 카드 정보 입력
4. 카드사 인증
5. 잠금 화면 설정 (필수)

### Samsung Pay 설정

1. Samsung Pay 앱 실행
2. 카드 등록
3. 지문/홍채 등록
4. 카드사 인증
5. 완료

## 사용 방법

### iPhone (Apple Pay)

**상점에서**

1. Face ID 또는 Touch ID 인증
2. 단말기에 폰 가까이 대기
3. 완료 신호 확인

**온라인/앱 내**

1. Apple Pay 선택
2. Face ID 또는 Touch ID
3. 결제 완료

### Android (Google Pay)

**상점에서**

1. 화면 켜기 (잠금 해제)
2. 단말기에 폰 가까이 대기
3. 진동 또는 소리로 확인

**소액 결제**

- 잠금 해제 불필요 (설정 가능)

### Samsung Pay

**상점에서**

1. 하단에서 위로 스와이프
2. 지문/홍채 인증
3. 단말기에 가까이 대기

## 해외 사용 시 주의사항

### 지원 여부 확인

⚠️ 출발 전 확인사항:

- 방문 국가에서 사용 가능한지
- 등록한 카드가 해외 결제 가능한지
- 카드사 해외 사용 승인 확인

### DCC 주의

💡 중요 팁: 모바일 결제도 DCC(원화 결제) 주의!

- 단말기에서 통화 선택 시
- 반드시 현지 통화 선택
- 원화 선택하면 손해

### 결제 한도

- 일부 국가는 소액만 가능
- 큰 금액은 PIN 번호 필요
- 국가별로 다름

## 국가별 모바일 결제

### 미국

- Apple Pay, Google Pay 가장 보편적
- 대부분의 상점에서 사용 가능
- 교통카드로도 사용 (뉴욕, 샌프란시스코 등)

### 일본

- Apple Pay로 Suica 등록 가능
- 편의점, 레스토랑에서 광범위하게 사용
- QR코드 결제 (PayPay, LINE Pay)도 많음

### 유럽

- Apple Pay, Google Pay 보편적
- 런던 지하철 모바일 결제 가능
- contactless 결제 일반화

### 중국

- 위챗페이, 알리페이 필수
- 외국인도 등록 가능 (여권 필요)
- Apple Pay, Samsung Pay도 가능하지만 제한적

### 동남아

- 싱가포르: 광범위하게 사용
- 태국: 주요 상점에서 사용 가능
- 베트남: 제한적, 현금 위주

## 중국 모바일 결제

### 위챗페이 (WeChat Pay)

**설정 방법**

1. 위챗 앱 다운로드
2. 회원 가입 (전화번호)
3. 해외 카드 등록
4. 실명 인증 (여권)

**사용처**

- 편의점, 레스토랑, 택시
- 거의 모든 곳에서 사용 가능

### 알리페이 (Alipay)

**설정 방법**

1. 알리페이 앱 다운로드
2. 해외 버전 선택
3. 카드 등록
4. 실명 인증

**특징**

- 관광지 많이 지원
- 영어 지원
- 환급 서비스

## 보안

### 안전한 이유

💡 모바일 결제가 카드보다 안전한 이유:

- 카드 번호 노출 안 됨
- 일회용 토큰 사용
- 생체 인증 필수
- 분실 시 원격 삭제 가능

### 분실 시 대처

**iPhone**

1. iCloud.com 접속
2. Find My iPhone
3. 분실 모드 활성화
4. 또는 원격 삭제

**Android**

1. Google Find My Device
2. 기기 잠금
3. 또는 데이터 삭제

**즉시 할 일**

- 카드사에 분실 신고
- 새 기기에 재등록

## 장단점 비교

### 모바일 결제 장점

- 빠르고 편리
- 지갑 필요 없음
- 보안 강력
- 분실 시 원격 잠금 가능
- 영수증 자동 저장

### 모바일 결제 단점

- 배터리 방전 시 사용 불가
- NFC 단말기 필요
- 모든 국가에서 사용 불가
- 소액만 가능한 경우 있음

## 팁

💡 해외 모바일 결제 꿀팁:

- 출발 전 미리 등록 및 테스트
- 보조 배터리 필수 휴대
- 카드 1-2장은 백업으로 소지
- 영수증은 앱에서 확인 가능
- 교통카드 기능 활용 (도쿄, 런던 등)
- 환율 우대 카드 등록

## 자주 묻는 질문

**Q: 인터넷 없이도 되나요?**
A: 네, NFC 결제는 오프라인도 가능합니다. 단, 등록 시에는 인터넷 필요.

**Q: 수수료가 있나요?**
A: 모바일 결제 자체는 무료입니다. 카드 수수료만 발생.

**Q: 어느 것을 써야 하나요?**
A: 아이폰은 Apple Pay, 안드로이드는 Google Pay 또는 Samsung Pay.

**Q: 중국에서 한국 카드로 되나요?**
A: 위챗페이, 알리페이에 한국 카드 등록 가능합니다.
