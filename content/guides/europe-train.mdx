## 유럽 기차 여행이란?

유럽은 기차로 여행하기 가장 좋은 대륙입니다. 빠르고 편안하며, 도시 중심부를 연결합니다.

### 기차 여행의 장점

- 도심에서 도심으로 직행
- 공항보다 편리
- 경치 감상 가능
- 와이파이, 콘센트 제공
- 좌석 넓음
- 환경 친화적

## 유레일 패스란?

유레일 패스는 유럽 33개국의 기차를 자유롭게 이용할 수 있는 패스입니다.

### 유레일 vs 인터레일

**유레일 패스 (Eurail Pass)**

- 유럽 외 거주자용
- 한국인은 유레일 구매

**인터레일 패스 (Interrail Pass)**

- 유럽 거주자용
- 우리는 해당 없음

## 유레일 패스 종류

### Global Pass (글로벌 패스)

33개국 모두 이용 가능

**유효기간 옵션**

- 15일 이내 4일
- 1개월 이내 5일
- 1개월 이내 7일
- 1개월 이내 10일
- 1개월 이내 15일
- 2개월 연속
- 3개월 연속

### One Country Pass (1개국 패스)

특정 국가만 집중 여행

**인기 국가**

- 스위스
- 이탈리아
- 프랑스
- 독일
- 스페인

### 가격 (2025년 기준)

**Global Pass 예시 (성인 2등석)**

- 1개월/5일: €257
- 1개월/7일: €303
- 1개월/10일: €384
- 2개월 연속: €578

**할인**

- 청소년 (12-27세): 25% 할인
- 어린이 (4-11세): 무료 (성인 1인당 최대 2명)
- 시니어 (60세 이상): 10% 할인

## 구매 방법

### 온라인 구매

**공식 사이트**

- eurail.com (영어)
- 한국 대리점

**장점**

- 할인 많음
- 편리
- 모바일 패스 가능

### 한국에서 구매

- 여행사
- 온라인 여행사
- 다소 비쌈

💡 팁: 공식 사이트에서 직접 구매가 가장 저렴

## 사용 방법

### 활성화 (Activation)

1. 첫 탑승 전 활성화
2. 역 창구 또는 앱에서
3. 여권 제시

### 모바일 패스

**Rail Planner 앱**

1. 앱 다운로드
2. 패스 등록
3. 여행 계획 추가
4. QR 코드 생성
5. 검표 시 제시

### 종이 패스

- 여행 일지 작성
- 날짜와 경로 기입
- 검표원에게 제시

## 예약 시스템

### 예약 필수 열차

⚠️ 유레일 패스로도 예약 필요:

- 고속열차 (TGV, ICE, AVE 등)
- 야간 열차
- 관광 열차
- 성수기 인기 노선

**예약 방법**

- 공식 사이트
- 역 창구
- 전화 예약

**예약 비용**

- €3-35 (열차마다 다름)
- 야간 열차: €40-60

### 예약 불필요 열차

- 독일 대부분 (IC, RE)
- 스위스 (대부분)
- 네덜란드
- 벨기에
- 지역 열차

💡 팁: 예약 없는 노선 위주로 계획하면 비용 절약

## 손익분기점

유레일 패스가 유리한지 계산해보세요.

### 계산 방법

1. 방문 도시 정하기
2. 개별 티켓 가격 조사
3. 유레일 패스 가격과 비교
4. 예약 비용 추가

### 예시

**파리-암스테르담-베를린-프라하 (10일)**

개별 티켓:

- 파리-암스테르담: €120
- 암스테르담-베를린: €90
- 베를린-프라하: €60
- 총: €270

유레일 패스 (1개월/5일): €257

→ 비슷하지만 유연성 고려하면 패스 유리

### 패스가 유리한 경우

- 여러 나라 방문
- 유연한 일정
- 장거리 많음
- 예약 불필요 노선

### 개별 티켓이 유리한 경우

- 1-2개 도시만
- 일정 확정
- 저가 항공 이용 병행
- 동유럽 위주 (저렴)

## 필수 앱

### Rail Planner

- 유레일 공식 앱
- 시간표 확인
- 모바일 패스
- 오프라인 사용 가능

### Trainline

- 유럽 기차 통합 검색
- 티켓 구매 가능
- 실시간 지연 정보

### DB Navigator

- 독일 철도 앱
- 정확한 시간표
- 플랫폼 정보

## 기차 타는 법

### 역에서

1. 출발 시간 및 플랫폼 확인
2. 플랫폼으로 이동
3. 차량 번호 확인 (1등석/2등석)
4. 탑승
5. 좌석 찾기 (예약 시)

### 좌석 예약 시스템

- 예약석: 좌석 위 표시 확인
- 자유석: 예약 표시 없으면 앉아도 됨
- 중간 구간만 예약된 경우도 있음

### 짐 보관

- 좌석 위 선반
- 객실 입구 짐 공간
- 큰 짐은 짐칸 (1등석 주로)

## 주의사항

### 파업

⚠️ 유럽은 파업이 잦습니다:

- 프랑스, 이탈리아 특히 주의
- 미리 확인
- 대체 경로 준비
- 보상 불가능 (패스 사용 일수 차감)

### 검표

- 무작위 검표
- 벌금 매우 비쌈 (€100+)
- 활성화 필수
- 여권 항상 소지

### 야간 열차

- 침대칸 예약 필수
- 비쌈 (€40-100)
- 숙박비 절약 가능
- 귀중품 주의

## 팁

💡 유레일 패스 활용 꿀팁:

- Rail Planner 앱 필수 설치
- 예약 불필요 노선 위주로
- 독일과 스위스는 예약 거의 불필요
- 야간 열차로 숙박비 절약
- 1등석 업그레이드 고려 (한산함)
- 역 라운지 이용 (1등석)
- 짐 미리 정리 (역 대기 시간 짧음)

## 1등석 vs 2등석

### 1등석

**장점**

- 좌석 넓고 편함
- 한산함
- 콘센트 많음
- 라운지 이용 가능

**단점**

- 비쌈 (50% 추가)

**추천**

- 장거리 여행
- 조용히 쉬고 싶을 때
- 업무할 때

### 2등석

**장점**

- 저렴
- 현지인과 교류
- 충분히 편함

**단점**

- 붐빌 수 있음
- 시끄러울 수 있음

**추천**

- 단거리 여행
- 배낭 여행
- 예산 여행

## 체크리스트

구매 전:

- ✓ 여행 일정 확정
- ✓ 손익분기점 계산
- ✓ 패스 종류 선택
- ✓ 청소년/시니어 할인 확인
- ✓ 할인 코드 검색

여행 전:

- ✓ Rail Planner 앱 설치
- ✓ 예약 필요 열차 확인
- ✓ 좌석 예약 완료
- ✓ 여권 유효기간 확인
- ✓ 모바일 패스 등록
- ✓ 오프라인 지도 다운로드

탑승 전:

- ✓ 패스 활성화
- ✓ 여행 일지 작성 (종이 패스)
- ✓ 플랫폼 확인
- ✓ 출발 시간 확인
- ✓ 여권 소지
