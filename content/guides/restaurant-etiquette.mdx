## 기본 레스토랑 에티켓

### 입장 및 착석

**대부분의 국가**

- 입구에서 직원이 안내할 때까지 대기
- "How many?" 질문에 인원수 대답
- 테이블 배정 받기
- 마음에 안 들면 변경 요청 가능

**예외: 캐주얼한 곳**

- 빈 테이블에 자유롭게 착석
- 셀프 서비스
- 카페, 푸드코트 등

### 주문하기

**순서**

1. 음료 주문 (와인, 물 등)
2. 에피타이저/전채 주문
3. 메인 요리 주문
4. 디저트는 메인 후 주문

**주문 팁**

- 메뉴판 펼쳐보기
- 모르는 메뉴는 질문하기
- 알레르기 미리 알리기
- "What do you recommend?" 추천 요청

## 나라별 식사 예절

### 일본

**젓가락 사용법**

- 음식을 찍지 않기
- 젓가락으로 가리키지 않기
- 그릇 위에 젓가락 놓지 않기
- 젓가락 받침 사용

**국물 요리**

- 그릇을 들고 마시기 OK
- 후루룩 소리 내도 됨
- 레이멘, 소바 등

**금기사항**

- 길거리에서 걸어다니며 먹기 금지
- 코 풀지 않기
- 전화 통화 자제

### 한국

**술자리 문화**

- 어른에게 술 받을 때 두 손으로
- 옆으로 고개 돌려 마시기
- 빈 잔에 따르기

**국물 요리**

- 숟가락 사용
- 그릇을 들지 않음
- 젓가락과 숟가락 동시 사용 금지

### 중국

**회전 테이블**

- 시계 방향으로 돌리기
- 음식은 자기 앞으로 와서 담기
- 다른 사람 앞의 음식 손대지 않기

**기타**

- 뼈, 가시는 테이블에 뱉어도 됨
- 접시에 조금 남기기 (배불렀다는 의미)
- 트림, 딸꾹질 OK

### 미국

**테이블 매너**

- 냅킨은 무릎 위에
- 팔꿈치 테이블 위에 올리지 않기
- 한 입 크기로 자르기
- 입에 음식 있을 때 말하지 않기

**팁 문화**

- 15-20% 필수
- 서비스가 나빠도 최소 10%
- 계산서에 팁란이 있음

### 프랑스

**식사 시간**

- 점심: 12:00-14:00
- 저녁: 19:30-22:00
- 빵은 손으로 뜯어 먹기
- 포크는 왼손, 나이프는 오른손

**와인**

- 직접 따르지 않기
- 직원이나 호스트가 따름
- 잔은 1/3만 채우기

**금기사항**

- 식사 중 전화 금지
- 큰 소리로 이야기하지 않기
- 손으로 먹지 않기

### 이탈리아

**파스타**

- 포크만 사용 (숟가락 X)
- 소량씩 돌돌 말아 먹기
- 접시에 조금 남기지 않기

**식사 순서**

- 안티파스토(전채)
- 프리모(파스타/리조또)
- 세콘도(메인)
- 돌체(디저트)
- 카페(에스프레소)

**커피**

- 카푸치노는 오전만
- 에스프레소는 식후에
- 라떼는 아침만

### 영국

**애프터눈 티**

- 스콘 먼저
- 잼 먼저, 크림 나중
- 샌드위치는 손으로
- 차는 우유 넣어 마시기

**펍 문화**

- 카운터에서 주문 및 결제
- 팁 불필요
- 자리는 선점

## 계산하기

### 계산 요청

**국가별 표현**

- 미국: "Check, please"
- 영국: "Bill, please"
- 프랑스: "L'addition, s'il vous plaît"
- 일본: "お会計お願いします" (오카이케이 오네가이시마스)
- 손으로 X 표시하기

### 계산 방법

**미국식**

- 테이블에서 카드 결제
- 팁 금액 직접 쓰기
- 영수증 2장 (고객용, 가맹점용)

**유럽식**

- 카운터 결제 또는 테이블 결제
- 단말기 직접 가져옴
- 팁 포함 여부 확인

**아시아식**

- 대부분 카운터 결제
- 테이블 번호 말하기
- 영수증 가져가기

### 더치페이

**미국/영국**

- "Separate checks, please"
- "Can we split the bill?"

**한국/일본**

- 1/n 계산기 사용
- 또는 각자 카운터에서 결제

## 금기사항 정리

### 절대 하지 말아야 할 것

- 큰 소리로 직원 부르기
- 손가락으로 직원 가리키기
- 테이블 두드리기
- 음식 사진만 찍고 안 먹기
- 음식 맛 없다고 큰 소리로 말하기

### 조심할 것

- 알레르기는 미리 알리기
- 종교적 금기 확인
- 드레스코드 확인
- 예약 시간 엄수
- 어린이 동반 가능 여부 확인

## 유용한 표현

### 영어

- "I have an allergy to..." (알레르기 있어요)
- "Is this spicy?" (매운가요?)
- "Can I have water?" (물 주세요)
- "This is delicious!" (정말 맛있어요!)
- "Can I get this to go?" (포장 가능한가요?)

### 일본어

- "すみません" (스미마센 - 저기요)
- "美味しい" (오이시이 - 맛있어요)
- "お水ください" (오미즈쿠다사이 - 물 주세요)
- "ごちそうさまでした" (고치소사마데시타 - 잘 먹었습니다)

## 드레스코드

### 파인 다이닝

- 남성: 재킷, 셔츠, 구두
- 여성: 원피스 또는 정장
- 반바지, 슬리퍼 금지
- 모자 벗기

### 캐주얼 레스토랑

- 청바지 OK
- 운동화 OK
- 단정한 차림

### 예약 시 확인

"Is there a dress code?"로 확인하세요.
