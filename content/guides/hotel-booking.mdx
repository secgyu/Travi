## 호텔 예약 사이트 비교

### 글로벌 예약 사이트

**Booking.com**

- 장점: 방대한 숙소, 무료 취소 많음
- 단점: 가격이 최저가 아닐 수 있음
- 추천: 유럽 여행

**Agoda**

- 장점: 아시아 숙소 많음, 가격 저렴
- 단점: 취소 수수료 높음
- 추천: 동남아 여행

**Hotels.com**

- 장점: 10박 적립 시 1박 무료
- 단점: 선택지가 적을 수 있음
- 추천: 장기 여행자

**Expedia**

- 장점: 항공+숙박 패키지 할인
- 단점: 단독 예약 시 비쌈
- 추천: 패키지 여행

### 직접 예약 vs 예약 사이트

**호텔 직접 예약**

- 베스트 프라이스 보장
- 포인트 적립
- 업그레이드 가능성
- 추가 요청 가능

**예약 사이트**

- 비교가 쉬움
- 포인트/쿠폰 활용
- 한꺼번에 관리

## 최저가 찾는 방법

### 1. 여러 사이트 비교

최소 3-4개 사이트를 비교하세요.

- Booking.com
- Agoda
- Hotels.com
- 호텔 공식 사이트

### 2. 메타 서치 엔진 활용

- Trivago
- 카약(Kayak)
- 호텔스컴바인

### 3. 예약 타이밍

**최적 예약 시기**

- 국내: 1-2주 전
- 해외: 2-3개월 전
- 성수기: 6개월 전

**요일별 팁**

- 평일 숙박: 목요일 예약
- 주말 숙박: 일요일 예약

### 4. 쿠폰 & 프로모션

- 첫 가입 할인
- 신용카드 제휴 할인
- 시즌 특가
- 조기 예약 할인
- 장기 투숙 할인

## 체크인/체크아웃 팁

### 체크인 전

- 얼리 체크인 가능 여부 문의
- 공항 픽업 서비스 확인
- 특별 요청사항 미리 전달

### 체크인 시

- 신분증, 예약 확인서 준비
- 보증금(Deposit) 안내 확인
- 객실 업그레이드 요청
- 조식 포함 여부 확인

### 체크아웃 시

- 미니바 사용 내역 확인
- 레이트 체크아웃 요청
- 짐 보관 서비스 활용

## 호텔 선택 기준

### 위치

가장 중요합니다!

- 대중교통 접근성
- 주요 관광지와의 거리
- 주변 편의시설
- 안전성

### 리뷰 확인

- 평점 7.5 이상 추천
- 최근 리뷰 우선 확인
- 사진 리뷰 참고
- 부정적 리뷰도 꼭 읽기

### 시설

- 와이파이 무료 제공
- 조식 포함 여부
- 수영장, 피트니스
- 세탁 서비스

## 취소 정책 주의사항

### 무료 취소

- 대부분 24-48시간 전까지
- 성수기는 더 일찍
- 특가 상품은 취소 불가 많음

### 노쇼(No-Show)

예약하고 나타나지 않으면:

- 첫날 숙박비 청구
- 또는 전액 청구
- 환불 불가

## 비용 절약 팁

### 1. 멤버십 활용

- 호텔 체인 멤버십 가입
- 포인트 적립 및 사용
- 회원 전용 특가

### 2. 장기 투숙 할인

- 3박 이상 할인
- 7박 이상 더 큰 할인
- 직접 협상 가능

### 3. 비수기 여행

- 50-70% 저렴
- 업그레이드 기회
- 여유로운 여행

### 4. 공유 숙소 고려

- Airbnb
- 게스트하우스
- 호스텔

## 체크리스트

예약 전 확인사항:

- ✓ 총 가격 확인 (세금 포함)
- ✓ 무료 취소 가능 여부
- ✓ 조식 포함 여부
- ✓ 주차 가능 여부 및 비용
- ✓ 체크인/체크아웃 시간
- ✓ 공항 셔틀 서비스
- ✓ 와이파이 무료 제공
- ✓ 추가 인원 요금
