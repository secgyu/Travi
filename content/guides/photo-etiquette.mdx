## 사진 촬영 기본 예절

여행지에서 사진을 찍을 때 지켜야 할 기본적인 매너입니다.

### 기본 원칙

- 사람 얼굴이 나오면 허락 받기
- 금지 표시 준수
- 플래시 사용 주의
- 다른 사람 방해하지 않기
- 사유지 침입 금지
- 위험한 행동 금지

## 촬영 금지 장소

### 절대 촬영 금지

⚠️ 촬영하면 법적 처벌:

**군사 시설**

- 공항 보안 구역
- 군사 기지
- 국경 검문소
- 경찰서 내부

**정부 시설**

- 대사관/영사관
- 법원 내부
- 교도소
- 정부 청사 (일부)

**사진 금지 표시가 있는 곳**

- 박물관 특정 전시물
- 미술관 작품
- 성당/사원 내부
- 공연장 (공연 중)

### 허락 필요한 곳

**상업 공간**

- 상점 내부
- 레스토랑
- 호텔 로비
- 카페

**종교 시설**

- 성당, 교회
- 사원, 모스크
- 절, 신사
- 묘지

**사람이 있는 곳**

- 재래시장
- 현지인 얼굴
- 어린이
- 공연자

## 나라별 촬영 문화

### 일본

**예의 중시**

- 사람 촬영 시 허락 필수
- 신사/사원 일부 금지
- 게이샤 무단 촬영 금지 (교토)
- 조용히 촬영

### 한국

**비교적 자유**

- 관광지 자유롭게 촬영
- 사람 얼굴은 허락 받기
- 군사 시설 주의
- 궁궐 내부 일부 금지

### 미국

**자유로운 편**

- 공공장소 대부분 가능
- 사유지 주의
- 어린이 촬영 민감
- 상업용은 허가 필요

### 중동

**매우 엄격**

- 여성 촬영 금지
- 모스크 내부 금지
- 정부 시설 엄격
- 허락 받고 촬영

### 유럽

**국가별 차이**

- 프랑스: 에펠탑 야경 상업용 금지
- 이탈리아: 성당 내부 대부분 금지
- 영국: 박물관 일부 허용
- 독일: 개인정보 보호 엄격

## 박물관/미술관 촬영

### 촬영 가능 여부 확인

💡 입장 전 확인사항:

- 티켓 구매 시 질문
- 안내 표지판 확인
- 직원에게 문의
- 웹사이트 사전 확인

### 촬영 규칙

**일반적 규칙**

- 플래시 금지 (대부분)
- 삼각대 금지
- 셀카봉 금지 (대부분)
- 특정 작품만 금지

**허용되는 경우**

- 플래시 없이
- 다른 관람객 방해 없이
- 빠르게
- SNS 공유 OK (개인 용도)

### 유명 박물관 정책

**루브르 (파리)**

- 대부분 촬영 가능
- 플래시 금지
- 모나리자 주변 혼잡

**대영박물관 (런던)**

- 대부분 촬영 가능
- 플래시 금지
- 무료 입장

**메트로폴리탄 (뉴욕)**

- 대부분 촬영 가능
- 특별 전시는 금지

**바티칸 박물관**

- 대부분 금지
- 시스티나 성당 엄격 금지

## 종교 시설 촬영

### 성당/교회

**일반적 규칙**

- 미사 중 촬영 금지
- 플래시 금지
- 조용히 촬영
- 안내 표지 확인

**유명 성당**

- 사그라다 파밀리아: 촬영 가능
- 성 베드로 대성당: 촬영 가능
- 노트르담: 촬영 가능

### 사원/절

**불교 사원**

- 대부분 허용
- 불상 발바닥 촬영 금지
- 스님 허락 받기
- 신발 벗고 입장

**힌두 사원**

- 일부 금지
- 허락 받고 촬영
- 내부 금지 많음

### 모스크

**이슬람 사원**

- 대부분 허용 (기도 시간 제외)
- 여성은 히잡 착용
- 신발 벗기
- 기도하는 사람 촬영 금지

## 사람 촬영 예절

### 허락 받는 법

**영어 표현**

- "May I take your photo?" (사진 찍어도 될까요?)
- "Can I take a picture with you?" (같이 찍어도 될까요?)
- "Is it OK to take photos?" (사진 찍어도 되나요?)

**제스처**

- 카메라 가리키며
- 물어보는 표정
- 엄지 척

### 거절당했을 때

- "No problem" (문제없어요)
- "Thank you anyway" (어쨌든 감사합니다)
- 웃으며 떠나기
- 절대 몰래 찍지 않기

### 팁 문화

**사진 촬영 팁**

- 민속 의상 입은 사람
- 거리 공연자
- 일부 관광지 현지인

$1-5 정도 적당

## 드론 촬영

### 드론 규제

⚠️ 국가별로 매우 다름:

**엄격한 국가**

- 프랑스 (파리 금지)
- 영국 (런던 금지)
- 인도 (거의 금지)
- 중동 (대부분 금지)

**허용적인 국가**

- 미국 (등록 필요)
- 캐나다
- 호주
- 뉴질랜드

### 드론 사용 주의

💡 드론 사용 전 확인:

- 현지 법규 확인
- 등록 필요 여부
- 비행 금지 구역 (공항, 정부 시설)
- 높이 제한
- 사생활 침해 주의
- 보험 가입

## 안전한 촬영

### 위험한 촬영 금지

⚠️ 절대 하지 마세요:

**위험한 장소**

- 철로 위
- 절벽 가장자리
- 교통 방해
- 야생동물 접근

**사고 사례**

- 셀카 사고 증가
- 절벽 추락
- 교통사고
- 동물 공격

### 안전 수칙

💡 안전한 촬영 팁:

- 주변 확인
- 안전한 장소
- 셀카봉 주의
- 발밑 확인
- 교통 신호 준수
- 위험 표지 준수

## SNS 공유 주의

### 공유 전 확인

**체크사항**

- 얼굴이 나온 사람 동의
- 위치 정보 (안전)
- 사유지 여부
- 저작권 (작품 등)

### 저작권

⚠️ 저작권 주의:

**촬영 금지 아님 = 상업 사용 가능은 아님**

- 에펠탑 야경 (조명 저작권)
- 미술 작품
- 건축물 일부
- 공연, 전시

개인 SNS는 대부분 OK, 상업용은 허가 필요

## 촬영 장비

### 기내 반입

**기내 수하물**

- 카메라 본체
- 렌즈
- 배터리 (리튬 100Wh 이하)
- 메모리 카드

**위탁 수하물**

- 삼각대 (일부 항공사 기내 가능)
- 배터리 금지 (리튬 배터리는 기내만)

### 현지 대여

- 렌즈 대여 가능
- 드론 대여 (규제 확인)
- GoPro 등 액션캠
- 수중 카메라

## 팁

💡 사진 촬영 꿀팁:

- 이른 아침/늦은 오후 (빛 좋음)
- 인파 피하기
- 현지인에게 촬영 부탁 (교류 기회)
- 백업 배터리 필수
- 메모리 카드 여분
- 클라우드 백업
- 방수 케이스 (해변, 우천 시)

## 체크리스트

촬영 전:

- ✓ 촬영 금지 표시 확인
- ✓ 사람 얼굴 나올 시 허락
- ✓ 플래시 사용 가능 여부
- ✓ 종교 시설 예의
- ✓ 안전 확인
- ✓ 드론 규제 확인
- ✓ 배터리 충전
- ✓ 메모리 공간 확인

공유 전:

- ✓ 사람 동의 확인
- ✓ 위치 정보 주의
- ✓ 저작권 확인
- ✓ 개인정보 모자이크
